## Development

To install for development, clone this repo and install with
[`yarn`](https://yarnpkg.com/):

```sh
# Clone
git clone git@github.com:Figedi/denon_menu && cd denon_menu

# Install the packages
yarn

# Start dev task
npm run dev
```
